Preparing for Installation

Download files here

Please watch out the completeness of these files!

  1. [android-rootfs.tar] size: 71464960, MD5: 326c18b344576a138d9c913927bb56b6
  2. [boot.tar] size: 6707200, MD5: 960257d17cbb72dd12d7a09636b76409
  3. [uImage.install] size: 4028346, MD5: 305b3a882ea9ddfe2ef8ff95cc48562a

There must be more than 1GB free space for internal media partition of Veer

Installation Steps

  1. Power up Veer，link USB with PC, and mount as a USB disk. Put android-rootfs.tar, boot.tar into the Veer USB disk.
  2. Restart Veer，then holding the volume up key，get into Recovery mode (there will be a USB arrow)

  3. In PC, get into the directory of uImage.install, open cmd and put this command:

    novacom boot mem:// < uImage.install

    You can follow this way "Install Bootloader"

    Put uImage.install into the c:\Program Files\Palm, Inc folder or wherever your novacom executable file installed, then open a command prompt on your computer and use the cd command to navigate to the folder you transferred the uImage.install to earlier (C:\Program Files\Palm, Inc for Windows).

    Then run the command

    novacom.exe boot mem:// < uImage.install

  4. There will be installing logs in the screen of Veer. After installing, it will restart automatically

  5. You will see the moboot multi-boot screen. Use volume key to select, and power button to confirm.
